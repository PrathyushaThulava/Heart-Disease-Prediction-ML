# Heart Disease Prediction Using Machine Learning

## Objective
Build and compare multiple machine learning models to predict heart disease.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Models Used
1. Logistic Regression
2. Random Forest Classifier
3. K-Nearest Neighbors

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score

## Visualizations
- Correlation Heatmap
- Feature Importance Chart
- Confusion Matrix

## Conclusion
Random Forest produced the best prediction performance and was selected as the final model.
